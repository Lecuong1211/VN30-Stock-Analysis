1. Title
# VN30 Stock Analysis
2. Project Overview

## Project Overview

This project analyzes historical stock price data of VN30 companies using Python.

The project focuses on exploratory data analysis, financial ratio analysis, and data visualization.
3. Problem Statement
## Problem Statement

This project aims to:

- Download historical stock data.
- Analyze stock performance.
- Calculate daily returns and volatility.
- Compare relationships among stocks.
- Visualize financial data.
4. Tech Stack
## Tech Stack

- Python
- pandas
- NumPy
- matplotlib
- seaborn
- vnstock
